# webshell

xterm + node-pty + websocket

## Installation   

```sh
npm install
```

## Build

### Develepment

```sh
npm run dev
```

### Production

```sh
npm run build
```

## Serve

```sh
npm run serve
```

Open your browser http://localhost:8999
